# Fancy Todo

## POST /todos
- Request body: 
    ```json
    {
        "title": "",
        "description": "",
        "status": "",
        "due_date": ""
    }.
    ```
- Response (accepted) :
    status code => 201,
    ```json
    {
        "id": "",
        "title": "",
        "description": "",
        "status": "",
        "due_date": ""
    }.
    ```
- Response (fail validation) :
    status code => 400,
    {
        "Validation Error": 
    }.
- Response (fail server) :
    status code => 500.

## GET /todos
- Response (accepted) : 
    status code => 200,
    [
        {
            "all data todos"
        }
    ].
- Response (fail) : 
    status code => 500.

## GET /todos/:id
- Response (accepted) : 
    status code => 200,
    {
        "data todo by id"
    }.
    
- Response (fail) : 
    status code => 404,
    {
        "error not found"
    }.
