# Fancy To-Do App
Fancy To-Do App is an application 
* RESTful endpoint for Todo's CRUD operation
* JSON formatted response

&nbsp;

## RESTful endpoints
### POST /register

> register new user

_Request Header_
```
not needed
```

_Request Body_
```json
{
    "userrname": "<string>",
    "email": "<string>",
    "password": "<string>"
}
```

_Response (201 - created)_
```json
{
    "id": "<given id by system>",
    "username": "<user's username>",
    "email": "<user's email>",
    "password": "<user's password>"
}
```

_Response (400 - bad request)_
```json
[
    {
        "column": "<error column from input>",
        "message": "<error message from system validation>"
    }
]
```

_Response (500 - internalServerError)_
```json
{
    "message": "internal server error"
}
```

### POST /login

> login user

_Request Header_
```
not needed
```

_Request Body_
```json
{
    "email": "<string>",
    "password": "<string>"
}
```

_Response (200 - OK)_
```json
{
    "access_token": "<access_token generated by system>"
}
```

_Response (401 - invalidEmailPassword)_
```json
{
    "message": "Invalid email / password"
}
```

_Response (404 - notFound)_
```json
{
    "message": "error not found"
}
```

_Response (500 - internalStatusError)_
```json
{
    "message": "internal server error"
}
```

### GET /todo
> get todo list

_Request Header_
```json
{

}
```

_Request Body_
```
not needed
```

_Response - (200 - OK)_
```json
[
  {
    "id": "<id from system>",
    "title": "<title from system>",
    "description": "<description from system>",
    "status": "<status from system>",
    "due_date": "<due date from system>",
    "UserId": "<todo's owner from system>",
    "createdAt": "<created date from system>",
    "updatedAt": "<last update date from system>"

  }
```

