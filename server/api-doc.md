# Fancy Todo app

## RESTful endpoints

### POST /todos

> Post new todo

_Request Header_
```
{
    "access_token": "<your access token>"
}
```

_Request Body_
```
{
    "title": "<title to get insert into>",
    "description": "<description to get insert into>",
    "status": <false when first created>,
    "due_date": "<due date to insert into>"
}
```

_Response (201)_
```
{
    "id": <given by system>,
    "title": "<posted title>",
    "description": "<posted description>",
    "status": <false>,
    "due_date": "<posted due date>",
    "createdAt": "<2020-03-20T07:15:12.149Z>",
    "updatedAt": "<2020-03-20T07:15:12.149Z>"
}
```

_Response (400)_
```
{
    "message": "due date must be greater or equal today"
}
```

_Response (500)_
```
{
    "message": "Internal Server Error"
}
```

### POST /signUp

> Post new User

_Request Body_
```
{
    "email": "<email to post to User>",
    "password": "<password to post to User>"
}
```

_Response (201)_
```
{
    "id": <given by the system>,
    "email": "<User email>"
}
```

_Response (400)_
```
{
    errors
}
```
